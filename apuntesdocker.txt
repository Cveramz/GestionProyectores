0. Crear proyecto en docker hub
 
1. docker volume prune en la carpeta raiz del proyecto
docker volume prune

2. cramos el volumen de la db:
docker volume create postgres-db-data

3.vemos los volumenes:
docker volume ls

4. ir a cada microservicio y apretar en maven (derecha) para poner clean y luego install

5. Luego crear cada dockerFile

6. En cada proyecto poner docker build -t cveramz/portingeso .
y luego docker push cveramz/portingeso
